Qn 1. 

function counter(){
        var counter = 0;
    
        function IncreaseCounter() {
            return counter += 1;
        };
    
        return IncreaseCounter;
    }
    
    var counter = counter();
    alert(counter());
    alert(counter());
    alert(counter());
    alert(counter());

Ans - 

Output will show pop-up screen in browser displaying counter value as 1 and in next call again a po
 screen will appear but this time counter value is updated to 2 and so on.This process is continue
 till the last call and every time counter value is increased by 1.
